## Cara Menggunakan

1. Buka aplikasi di browser.
2. Klik **Register** untuk membuat akun baru, atau **Login** jika sudah punya akun.
3. Setelah login, klik **Booking** untuk membuat janji servis.
4. Pilih tanggal dan layanan yang diinginkan, lalu klik **Buat Booking**.
5. Lihat riwayat booking kamu di menu **Booking Saya**.
